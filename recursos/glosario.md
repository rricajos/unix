# Glosario de Terminos

## A

- **ACL (Access Control List)**: Lista de control de acceso. Permisos extendidos mas alla del modelo rwx.
- **APT (Advanced Package Tool)**: Gestor de paquetes de Debian y derivados.

## B

- **BIOS (Basic Input/Output System)**: Firmware basico de la placa base, previo a UEFI.
- **Bootloader**: Cargador de arranque. Programa que inicia el sistema operativo (GRUB, LILO).

## C

- **CIDR (Classless Inter-Domain Routing)**: Notacion de direccionamiento IP sin clases (ej: 192.168.1.0/24).
- **Cron**: Demonio para programar tareas periodicas.
- **CVE (Common Vulnerabilities and Exposures)**: Identificador estandar para vulnerabilidades.

## D

- **Daemon (Demonio)**: Proceso que se ejecuta en segundo plano.
- **DMA (Direct Memory Access)**: Acceso directo a memoria por parte de dispositivos de hardware.
- **DNS (Domain Name System)**: Sistema de resolucion de nombres de dominio.
- **DPKG**: Gestor de paquetes de bajo nivel en Debian.

## E

- **ESP (EFI System Partition)**: Particion de sistema EFI, necesaria para UEFI.

## F

- **FHS (Filesystem Hierarchy Standard)**: Estandar que define la estructura de directorios en Linux.
- **Firewall**: Cortafuegos. Sistema que filtra trafico de red.

## G

- **GPT (GUID Partition Table)**: Esquema de particionado moderno que reemplaza MBR.
- **GRUB (Grand Unified Bootloader)**: Cargador de arranque estandar en Linux.

## I

- **Inode**: Estructura de datos que almacena metadatos de un archivo en el sistema de archivos.
- **IRQ (Interrupt Request)**: Solicitud de interrupcion enviada por hardware al procesador.
- **IPC (Inter-Process Communication)**: Comunicacion entre procesos.

## K

- **Kernel**: Nucleo del sistema operativo. Gestiona hardware y recursos del sistema.

## L

- **LVM (Logical Volume Manager)**: Gestor de volumenes logicos para almacenamiento flexible.
- **LUKS (Linux Unified Key Setup)**: Estandar de cifrado de disco en Linux.

## M

- **MBR (Master Boot Record)**: Registro maestro de arranque. Esquema de particionado clasico (max 2TB).
- **MTA (Mail Transfer Agent)**: Agente de transferencia de correo (Postfix, Sendmail).
- **Modulo del kernel**: Codigo que se carga/descarga dinamicamente en el kernel.

## N

- **NFS (Network File System)**: Sistema de archivos en red.
- **NAT (Network Address Translation)**: Traduccion de direcciones de red.

## P

- **PID (Process ID)**: Identificador numerico de un proceso.
- **Pipe (Tuberia)**: Mecanismo para conectar la salida de un comando con la entrada de otro (`|`).
- **PAM (Pluggable Authentication Modules)**: Sistema modular de autenticacion.

## R

- **RAID**: Arreglo redundante de discos independientes.
- **RPM (Red Hat Package Manager)**: Formato de paquetes de Red Hat y derivados.
- **Runlevel (Nivel de ejecucion)**: Estado operativo de un sistema init (0-6).

## S

- **Shell**: Interprete de comandos (bash, zsh, sh).
- **SUID/SGID**: Permisos especiales que ejecutan con privilegios del propietario/grupo.
- **Syslog**: Sistema de registro centralizado.
- **Systemd**: Sistema init y gestor de servicios moderno.

## T

- **Target (systemd)**: Equivalente a los runlevels en systemd. Agrupa unidades.

## U

- **UEFI (Unified Extensible Firmware Interface)**: Reemplazo moderno de BIOS.
- **Udev**: Gestor de dispositivos dinamico del kernel Linux.
- **UID/GID**: Identificadores numericos de usuario y grupo.

## V

- **VPN (Virtual Private Network)**: Red privada virtual.

## X

- **X11 / Xorg**: Sistema de ventanas para interfaces graficas en Unix/Linux.
