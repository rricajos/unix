# Libros Recomendados

## LPIC-1

| Libro | Autor | Notas |
|-------|-------|-------|
| LPIC-1 Linux Professional Institute Certification Study Guide (5th ed.) | Christine Bresnahan, Richard Blum | Referencia principal para 101-500 y 102-500 |
| The Linux Command Line (2nd ed.) | William Shotts | Excelente para dominar la terminal. Disponible gratis en linea |
| How Linux Works (3rd ed.) | Brian Ward | Entiende como funciona Linux por debajo |

## LPIC-2

| Libro | Autor | Notas |
|-------|-------|-------|
| LPIC-2 Linux Professional Institute Certification Study Guide | Christine Bresnahan, Richard Blum | Guia oficial de estudio |
| Linux Administration: A Beginner's Guide (8th ed.) | Wale Soyinka | Buen complemento para servicios de red |
| UNIX and Linux System Administration Handbook (5th ed.) | Evi Nemeth et al. | La biblia de la administracion de sistemas |

## LPIC-3 / Seguridad

| Libro | Autor | Notas |
|-------|-------|-------|
| Linux Security Cookbook | Daniel Barrett et al. | Recetas practicas de seguridad |
| Mastering Linux Security and Hardening (3rd ed.) | Donald Tevault | Seguridad y hardening paso a paso |

## Hacking / Pentesting

| Libro | Autor | Notas |
|-------|-------|-------|
| The Hacker Playbook 3 | Peter Kim | Metodologia practica de pentesting |
| Penetration Testing | Georgia Weidman | Intro completa al pentesting |
| Linux Basics for Hackers | OccupyTheWeb | Linux enfocado a seguridad ofensiva |
| RTFM: Red Team Field Manual | Ben Clark | Referencia rapida de comandos ofensivos |
| BTFM: Blue Team Field Manual | Alan White, Ben Clark | Referencia rapida para defensores |

## Recursos gratuitos en linea

- [OverTheWire Wargames](https://overthewire.org/wargames/) - Practica con retos
- [Cybrary](https://www.cybrary.it/) - Cursos de IT y ciberseguridad
- [LPI Learning Materials](https://learning.lpi.org/) - Material oficial del LPI
