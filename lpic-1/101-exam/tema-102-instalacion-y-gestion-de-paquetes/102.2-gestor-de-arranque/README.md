# 102.2 - Instalar un gestor de arranque

## Peso: 2

## Objetivo del examen

Seleccionar, instalar y configurar un gestor de arranque. Esto incluye proporcionar una ubicacion de arranque alternativa y opciones de arranque de respaldo (por ejemplo, usando un cargador de arranque como GRUB Legacy o GRUB2).

## Conocimientos clave

- Proporcionar ubicaciones de arranque alternativas y opciones de arranque de respaldo
- Instalar y configurar un cargador de arranque como GRUB2
- Realizar cambios basicos de configuracion en GRUB2
- Interactuar con el cargador de arranque (menu de GRUB)
- Comprender los archivos de configuracion de GRUB2 y su estructura
- Conocimiento basico de GRUB Legacy (menu.lst)
- Recuperacion del sistema desde el menu de GRUB

## Archivos, terminos y utilidades

- `/boot/grub/grub.cfg` o `/boot/grub2/grub.cfg`
- `/etc/default/grub`
- `/etc/grub.d/`
- `grub-install`
- `grub-mkconfig`
- `update-grub`
- `grub2-install` (Red Hat)
- `grub2-mkconfig` (Red Hat)
- MBR (Master Boot Record)
- UEFI
- GRUB2
- GRUB Legacy
