# 104.5 Gestionar permisos y propiedad de archivos

## Objetivo
El candidato debe ser capaz de controlar el acceso a archivos mediante el uso adecuado de permisos y propiedades. Incluye permisos de acceso en archivos regulares y especiales, asi como directorios. Tambien incluye modos de acceso como SUID, SGID y sticky bit, el uso de los campos de grupo para otorgar acceso a grupos de trabajo, umask, y el uso de los permisos adecuados para cada tipo de archivo.

## Peso
**3**

## Conocimientos clave
- Gestionar permisos de acceso en archivos regulares y especiales, asi como directorios
- Usar modos de acceso como SUID, SGID y sticky bit para mantener la seguridad
- Saber cambiar la mascara de creacion de archivos (umask)
- Usar el campo de grupo para otorgar acceso a miembros del grupo

## Archivos y utilidades clave
| Recurso | Descripcion |
|---------|-------------|
| `chmod` | Cambiar permisos de archivos y directorios |
| `chown` | Cambiar propietario y grupo de archivos |
| `chgrp` | Cambiar grupo de archivos |
| `umask` | Establecer mascara de creacion de archivos |
| `ls -l` | Ver permisos en formato largo |
