# 103.7 - Buscar texto con expresiones regulares

## Peso: 3

## Objetivo del examen
Manipular archivos y datos de texto usando expresiones regulares. Incluye crear expresiones regulares simples que contengan varios elementos de notacion, asi como comprender las diferencias entre expresiones regulares basicas y extendidas. Tambien incluye el uso de herramientas de expresiones regulares para realizar busquedas en un sistema de archivos o en el contenido de archivos.

## Conocimientos clave
- Crear expresiones regulares simples que contengan varios elementos de notacion
- Comprender las diferencias entre expresiones regulares basicas y extendidas
- Comprender los conceptos de caracteres especiales, clases de caracteres, cuantificadores y anclas
- Usar herramientas de expresiones regulares para realizar busquedas en archivos y datos
- Usar expresiones regulares para borrar, cambiar y sustituir texto

## Archivos, terminos y utilidades
- `grep`
- `egrep`
- `fgrep`
- `sed`
- `regex(7)`
