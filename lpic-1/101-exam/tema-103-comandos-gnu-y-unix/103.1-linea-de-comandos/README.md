# 103.1 - Trabajar en la linea de comandos

## Peso: 4

## Objetivo del examen
Interactuar con shells y comandos usando la linea de comandos. Incluye el uso de comandos y secuencias de comandos simples, el uso y modificacion del entorno de shell, el uso del historial de comandos y la invocacion de comandos dentro y fuera de la ruta definida.

## Conocimientos clave
- Usar comandos simples y secuencias de comandos en la linea de comandos
- Usar y modificar el entorno del shell, incluyendo definir, referenciar y exportar variables de entorno
- Usar y editar el historial de comandos
- Invocar comandos dentro y fuera de la ruta definida (PATH)
- Usar paginas man, info y otros sistemas de documentacion
