# 103.8 - Edicion basica de archivos

## Peso: 3

## Objetivo del examen
Editar archivos de texto usando vi. Este objetivo incluye la navegacion en vi, los modos basicos de vi, insertar, editar, borrar, copiar y buscar texto. Tambien incluye el conocimiento de otros editores comunes y la configuracion del editor predeterminado.

## Conocimientos clave
- Navegar en un documento usando vi
- Usar los modos basicos de vi (normal, insercion, comando)
- Insertar, editar, borrar, copiar y buscar texto en vi
- Conocer otros editores comunes de texto (nano)
- Configurar el editor predeterminado

## Archivos, terminos y utilidades
- `vi`
- `/`, `?`
- `h`, `j`, `k`, `l`
- `i`, `o`, `a`
- `d`, `p`, `y`, `dd`, `yy`
- `ZZ`, `:w!`, `:q!`, `:e!`
- `:!`
- `nano`
