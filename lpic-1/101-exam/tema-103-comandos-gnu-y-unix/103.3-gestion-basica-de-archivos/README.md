# 103.3 - Gestion basica de archivos

## Peso: 4

## Objetivo del examen
Usar los comandos basicos de Linux para gestionar archivos y directorios. Incluye copiar, mover, eliminar archivos y directorios, copiar multiples archivos simultaneamente, buscar archivos recursivamente, usar patrones de globbing, y utilizar herramientas de archivado y compresion.

## Conocimientos clave
- Copiar, mover y eliminar archivos y directorios de forma individual
- Copiar multiples archivos y directorios de forma recursiva
- Eliminar archivos y directorios de forma recursiva
- Usar comodines simples y avanzados en comandos
- Usar find para buscar archivos basandose en tipo, tamanho, fecha y permisos
- Uso de tar, cpio y dd

## Archivos, terminos y utilidades
