# Unix - Preparacion LPIC & Hacking Vault

Repositorio de estudio para las certificaciones **LPIC-1, LPIC-2 y LPIC-3** del Linux Professional Institute, y base de conocimientos de **seguridad ofensiva y defensiva**.

## Estructura del repositorio

```
unix/
├── lpic-1/          Certificacion LPIC-1 (Examenes 101 + 102)
├── lpic-2/          Certificacion LPIC-2 (Examenes 201 + 202)
├── lpic-3/          Certificacion LPIC-3 (300, 303, 305, 306)
├── hacking-vault/   Base de conocimientos de seguridad
├── recursos/        Glosario, enlaces, cheatsheets
└── assets/          Plantillas y recursos compartidos
```

## Como usar este repositorio

1. **Navega por certificacion**: Cada directorio `lpic-X/` contiene los temas organizados por examen
2. **Estudia teoria**: Lee los archivos en `notas/` de cada subtema
3. **Practica**: Ejecuta los scripts en `scripts/` en un entorno de pruebas
4. **Autoexaminate**: Responde las preguntas en `ejercicios/`
5. **Aplica**: Usa el `hacking-vault/` para ver aplicaciones practicas de seguridad
6. **Marca progreso**: Edita este README marcando `[x]` los temas completados
