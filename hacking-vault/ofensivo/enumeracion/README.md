# Enumeracion

Descubrimiento detallado de servicios, usuarios, recursos compartidos y vulnerabilidades.

## Contenido

- [notas/](notas/) - Apuntes sobre enumeracion de servicios (SMB, LDAP, HTTP, DNS, SNMP)
- [scripts/](scripts/) - Scripts de enumeracion automatizada
- [herramientas/](herramientas/) - Guias de herramientas (enum4linux, gobuster, nikto, ldapsearch)

## Temas principales

- Enumeracion de servicios web (directorios, vhosts, APIs)
- Enumeracion SMB/CIFS
- Enumeracion LDAP y Active Directory
- Enumeracion DNS (transferencia de zona, brute force)
- Enumeracion SNMP

## Temas LPIC relacionados

- **LPIC-2 207**: Servidor de nombres de dominio (DNS)
- **LPIC-2 208**: Servicios web (Apache, Nginx)
- **LPIC-2 209**: Comparticion de archivos (Samba, NFS)
- **LPIC-2 210**: Gestion de clientes de red (LDAP, PAM)
