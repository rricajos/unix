# Seguridad Ofensiva (Red Team)

## Metodologia general

1. **Reconocimiento** - Recopilar informacion sobre el objetivo
2. **Enumeracion** - Descubrir servicios, usuarios, vulnerabilidades
3. **Explotacion** - Obtener acceso al sistema
4. **Post-Explotacion** - Mantener acceso, escalar privilegios, pivotar
