# Hardening

Configuracion segura de sistemas y servicios Linux para reducir la superficie de ataque.

## Contenido

- [notas/](notas/) - Apuntes sobre hardening de kernel, servicios, usuarios y red
- [scripts/](scripts/) - Scripts de automatizacion de hardening y auditoria

## Temas principales

- CIS Benchmarks para Linux
- Hardening del kernel (sysctl, modulos)
- Gestion segura de usuarios y permisos
- Configuracion segura de SSH
- Deshabilitacion de servicios innecesarios
- SELinux y AppArmor
- Auditoria con Lynis y OpenSCAP

- **LPIC-3 303 332.1**: Seguridad del host endurecido
- **LPIC-1 110.1**: Tareas de administracion de seguridad
- **LPIC-1 107.1**: Gestion de cuentas de usuario y grupo
