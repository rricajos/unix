# Laboratorios Practicos

## Requisitos del entorno
- VirtualBox o VMware
- Vagrant (opcional, para automatizar)
- Docker (para labs ligeros)
- Minimo 8GB RAM, 50GB disco libre

## Estructura
- [vagrant/](vagrant/) - Vagrantfiles para montar labs
- [docker/](docker/) - Dockerfiles para labs ligeros
- [escenarios/](escenarios/) - Labs guiados paso a paso
