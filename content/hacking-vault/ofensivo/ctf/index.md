# CTF y Writeups

## Metodologia CTF

1. Reconocimiento y escaneo inicial
2. Enumeracion de servicios
3. Buscar exploit o vulnerabilidad
4. Obtener shell de usuario
5. Escalar privilegios a root
6. Documentar todo el proceso
