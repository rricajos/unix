# Firewalls y Filtrado

Configuracion y gestion de firewalls en Linux para filtrado de paquetes y control de trafico.

## Contenido

- [notas/](notas/) - Apuntes sobre iptables, nftables, firewalld y arquitecturas de red
- [scripts/](scripts/) - Scripts de configuracion de reglas y automatizacion de firewalls

## Temas principales

- iptables: tablas, cadenas, reglas, NAT, MASQUERADE
- nftables: sintaxis moderna, migracion desde iptables
- firewalld: zonas, servicios, reglas enriquecidas
- Arquitecturas de red seguras (DMZ, segmentacion)
- Fail2ban: proteccion contra fuerza bruta

- **LPIC-3 303 334.3**: Filtrado de paquetes
- **LPIC-3 303 334**: Seguridad de red
