# 104.7 Encontrar archivos del sistema y su ubicacion correcta

## Objetivo
El candidato debe conocer bien las ubicaciones correctas de los archivos segun el FHS (Filesystem Hierarchy Standard), incluyendo la ubicacion tipica de archivos y conocer como encontrar archivos y comandos en un sistema Linux.

## Peso
**2**

## Conocimientos clave
- Comprender las ubicaciones correctas de archivos bajo el FHS
- Encontrar archivos y comandos en un sistema Linux
- Conocer la ubicacion y el proposito de archivos y directorios importantes definidos en el FHS

## Archivos y utilidades clave
| Recurso | Descripcion |
|---------|-------------|
| `find` | Buscar archivos por criterios diversos |
| `locate` / `mlocate` | Buscar archivos usando base de datos indexada |
| `updatedb` | Actualizar base de datos de locate |
| `whereis` | Localizar binario, fuente y man de un comando |
| `which` | Mostrar ruta completa de un ejecutable |
| `type` | Identificar tipo de comando (builtin, alias, etc.) |
| `/etc/updatedb.conf` | Configuracion de updatedb |
