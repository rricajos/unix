# 104.2 Mantener la integridad de los sistemas de archivos

## Objetivo
El candidato debe ser capaz de mantener un sistema de archivos estandar, asi como los datos adicionales asociados a un sistema de archivos con journaling. Incluye verificar, monitorear el espacio libre y reparar problemas simples.

## Peso
**2**

## Conocimientos clave
- Verificar la integridad de los sistemas de archivos
- Monitorizar el espacio libre y los inodos
- Reparar problemas simples de sistemas de archivos

## Archivos y utilidades clave
| Recurso | Descripcion |
|---------|-------------|
| `df` | Mostrar uso de espacio en disco |
| `du` | Estimar uso de espacio de archivos/directorios |
| `fsck` | Verificar y reparar sistemas de archivos |
| `e2fsck` | Verificar/reparar sistemas ext2/ext3/ext4 |
| `xfs_repair` | Reparar sistemas XFS |
| `tune2fs` | Ajustar parametros de sistemas ext |
| `dumpe2fs` | Mostrar informacion detallada de sistemas ext |
| `xfs_info` | Mostrar informacion de sistemas XFS |
| `mke2fs` | Crear sistemas de archivos ext |
