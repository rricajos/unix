# 110.1 Realizar tareas de administracion de seguridad

## Objetivo
Revisar la configuracion del sistema para garantizar la seguridad del host de acuerdo con las politicas de seguridad locales.

## Peso
3

## Conocimientos clave
- Auditar un sistema para encontrar archivos con SUID/SGID
- Establecer o cambiar contrasenas de usuario y la informacion de caducidad
- Ser capaz de usar nmap y netstat para descubrir puertos abiertos
- Configurar limites de recursos del sistema para usuarios
- Determinar que usuarios han iniciado sesion o estan conectados
- Configuracion y uso basico de sudo

## Archivos, terminos y utilidades
- `find`
- `passwd`
- `fuser`
- `lsof`
- `nmap`
- `chage`
- `netstat`
- `sudo`
- `/etc/sudoers`
- `su`
- `usermod`
- `ulimit`
- `who`, `w`, `last`
- `/etc/security/limits.conf`
